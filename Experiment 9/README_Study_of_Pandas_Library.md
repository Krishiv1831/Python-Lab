## ğŸ“˜ Theory (Operations in Pandas)

### 1. Creating a Series

A Series is a one-dimensional labeled array.
Example: `pd.Series([10,20,30])`

### 2. Creating a DataFrame

A DataFrame is a two-dimensional table.
Example: `pd.DataFrame(data)`

### 3. Accessing Data

Using `df["column"]` and `df.loc[index]`

### 4. Adding Columns

Using `df["NewColumn"] = value`

### 5. Deleting Columns

Using `df.drop()`

### 6. Statistical Operations

Using `mean()`, `mode()`, `min()`, `max()`
